Privacy Policy - LetaTab

Last updated: July 20, 2026

LetaTab is a browser extension, available for Firefox as well as Chrome, Edge, Brave, and other Chromium based browsers, that lets you set your own New Tab page and opens it every time you start a new tab.

Summary

This extension does not collect, transmit, sell, or share any personal information. There is no analytics, no crash reporting, no telemetry, and no update-check pings of any kind. The extension never contacts any server, including one of its own. The only data it stores is the web address you choose for your New Tab page and whether the extension is turned on or off, and that data never leaves your browser. On Firefox, this is also declared directly in the extension's manifest as "none" under Firefox's built in data collection disclosure.

What data is stored, and where

The extension stores exactly two values, using the browser's own built in sync storage API (called storage.sync, exposed as browser.storage.sync on Firefox and chrome.storage.sync on Chrome, Edge, and Brave):

  • The New Tab web address you have chosen
  • Whether the extension is enabled or disabled

This storage is provided by the browser itself, not by us. If you are signed in to your browser or to Firefox Sync, the browser vendor may sync this small setting across your own devices the same way it syncs your bookmarks. If you are not signed in, the extension still works the same way on that one device, the setting simply stays local until you sign in. The extension developer never receives, sees, or has access to this data on any server, because no such server exists for this extension.

What the extension does

LetaTab replaces the New Tab page shown by your browser. Every time you open a new tab, it opens the web address you have chosen in the popup, once you've chosen one. It does not read, modify, or interact with the content of any other page, tab, or site, and it does not request access to any website. It only opens the one address you picked, exactly like typing that address into the address bar yourself. Until you choose an address, the extension does not open anywhere on its own, it shows a short on-page message instead.

Link to letalab.eu

The extension's popup includes a plain link to letalab.eu, which opens in a new browser tab. The link itself carries no tracking script or tracking parameters. Clicking it is no different from typing the address yourself. Once you're on that site, its own privacy practices apply, separately from this extension.

Third parties

We do not share data with any third party, because we do not collect any data to share. The extension embeds no third-party analytics, advertising, or tracking scripts, and loads no remote code. Every file that runs is shipped inside the extension package itself.

Permissions

The extension requests exactly one permission, storage, used only to remember the New Tab address you choose and whether the extension is enabled. Nothing broader is requested: no host permissions, no tabs, no cookies, no webRequest, and no access to the content of any website.

Security

The extension's full source ships inside the package you install. Nothing is downloaded or evaluated at runtime. On Firefox, every release is also verified and signed by Mozilla before it can be installed permanently, which includes an independent review of the source. We welcome responsible security review. If you find a problem, please report it privately and with enough detail to reproduce it, via the contact below.

Children's privacy

The extension is not directed at children and does not knowingly collect information from anyone, regardless of age, since it does not collect information at all.

Changes to this policy

If this extension's data practices ever change, this page will be updated before the change takes effect, and the "Last updated" date above will be revised accordingly.

Contact

Questions about this policy, the extension, or a security report can be sent via letalab.eu.
